Меню предназначено для настройки свойств ресурса.

Интерфейс данного экрана относится к стандартной графической форме. Подробное описание формы приведено в разделе 1.1. Стандартная графическая форма «dict».

Для настройки параметров свойства используются следующие поля:

Название*

Поле определяет уникальное название свойства.

Тип*

Определяет тип свойства. Описание приведено в разделе 7.4. Типы свойств ресурса.

В поле содержится раскрывающийся список.

Длина

Поле определяет размер значения свойства в байтах.

Замечание

Для свойства типа Дата в данном поле указывается числовое значение, определяющее формат даты.

  • 1 - DD.MM.YY

  • 2 - DD.MM.YYYY

  • 3 - DD.MM.YYYY HH24:MI

  • 4 - MM.YY

Точность

В случае числового типа значения свойства этот атрибут определяет количество знаков после десятичной точки.

Маска

В случае если свойство ресурса связано с маской тарификатора, то указание маски тарификатора для свойства является сигналом для включения сервиса.

Значение по умолчанию

Набор функций, которые формируют значение свойства по умолчанию.

Список?

Выбранный флажок определяет образование списка. Это означает, что значение свойства включается в общий список, из которого оператор при подключении выбирает нужное значение.

RADIUS-атрибут

Значение свойства является значением RADIUS-атрибута. Из раскрывающегося списка необходимо выбрать необходимый атрибут.

Уровень*

Определяет уровень доступа к свойству т. е. права на внесение, изменение, удаление значения свойства. Может принимать значения:

  • Административный;

  • Пользовательский;

  • Пользовательский (RO);

  • Операторский;

  • Операторский (RO).

Рег. выражение

Регулярное выражение для синтаксического анализа значения свойства.

Наследуемое

Значение свойства наследуется из родительского подключения

Отображать*

Поле определяет способ отображения значения свойства в момент подключения ресурса. Может принимать значения:

  • Все;

  • Значение;

  • Ничего;

  • При создании.

Группа*

Для свойства обязательно указание групповой принадлежности. .

Russian/English/...

Название свойства на языках представленных в системе.

При создании свойства доступна ссылка Свойства/Сервисы , перейдя по которой можно получить список ресурсов, которому принадлежит свойство.

После заполнения полей Значение свойства и Значение по умолчанию становятся доступными следующие ссылки:

  • Значение свойства - определяет значение данного свойства;

  • Значение по умолчанию - определяет значение по умолчанию данного свойства.

Ссылка Аудит становится доступной после изменений, вносимых в параметры свойства. В аудите отражается история изменения свойства.
